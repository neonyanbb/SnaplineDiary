In the previous article we focused on why to record. This one focuses on a single practical task: how to create one new entry from start to finish.

Creating entries in Snapline Diary is already straightforward. This walkthrough simply makes the full flow explicit, so you can quickly understand each step and what it solves.

1. Two entry points to create a new entry

You can start from either place:

  1. Home + button: best when you actively organize photos (for example, lecture notes or travel memories).
  2. System Share Extension: best when you are in another app and want to save content immediately.

Different paths, same result: both create one entry and send it into your timeline on Home.

Tip: the Share Extension currently accepts images only. If your share selection contains video, Snapline Diary may not appear in the share sheet. Select images only, then share again.

2. Inside the page: five practical steps

1) Add photos

Most users work faster by adding photos first, then writing notes. You can pick from Photos and keep adding more in the same entry. The thumbnail strip helps confirm what has already been collected.

2) Choose cover (drag to reorder)

The image strip supports drag reorder. The first image becomes the cover.

This is useful when one entry has multiple images: move the most representative image to position one for easier recognition on Home.

3) Write notes (supported formats)

  • Multi-line plain text with line breaks;
  • Emoji supported (for example 🎯 and ✈️);
  • Pasted links are detected and can be opened from detail view;
  • No Markdown rich-text formatting in this editor;
  • Tags are managed separately from body text.

The focus is stability for capture, revisit, and source lookup.

4) Add tags (with recommendations)

You can add tags in two ways:

  • Type tags manually (space/enter submits continuously);
  • Tap recommended tags to add with one click.

Tip: you do not need #. Type the tag word directly, then press space or enter to generate the color chip.

Recommendations come from your current note keywords and your existing tag history, so suggestions stay personal and relevant.

5) Use quick edit buttons (Tag / Paste)

  • Tag: moves focus to tag input immediately;
  • Paste: reads clipboard and inserts content into note body.

These two buttons reduce interaction friction in fast capture scenarios.

3. What happens after saving

After you tap save, the entry is written locally and appears on Home immediately (usually near the top of today’s group).

Entries from Share Extension and Home + are unified into one timeline.

4. Real use scenarios

Scenario A: save useful video content instantly

  1. Take screenshots in a video app;
  2. Share to Snapline Diary;
  3. Paste transcript points and source link in notes;
  4. Add 2-4 tags;
  5. Save and retrieve later by timeline or tag.

This turns “watched” into “captured with evidence”.

Scenario B: organize systematic materials from Home

  • Group lecture key photos into one entry;
  • Group same-theme travel photos into one entry;
  • Drag a representative cover, add summary notes, then apply tags.

Later you can review by time on Home, or by topic via tags.

5. Wrap-up

The core of creating an entry is not another input screen, but one reliable flow:

Capture from two entry points → choose photos and cover → write note and links → add tags and recommendations → save back to timeline.
